## Fehler in der Aufbauanleitung von Carten
In der Anleitung ist bei der Montageinstruktion für das Servohorn ein Fehler.
Es wird gezeigt, dass das Kugelgelenk vorne auf das Servohorn geschraubt werden soll.
Das ist aber nicht der Fall. Das Kugelgelenk muss hinten an das Servohorn geschraubt werden.
